### Tips for Success as an International Barrister
- Highlight your Nigerian/common law qualification, any bar admission, research/drafting experience, and English proficiency in profiles.
- Start with freelance platforms like Lawclerk or Axiom to build remote portfolio and reviews.
- No upfront fees — legitimate roles never ask for payment.
- Combine platforms and apply consistently. Task volume can be higher with strong research skills or additional languages.
- For AML/KYC: Free online courses (e.g., on FATF or basic compliance) can strengthen applications.
